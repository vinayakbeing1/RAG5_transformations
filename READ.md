# Sports Impact on Life and Health

## Overview

This project analyzes the impact of sports participation on physical health, mental well-being, social development, and overall lifestyle through survey-based research. The analysis is performed using Python and various data visualization techniques.

## Objectives

* Study the effects of sports on physical fitness.
* Analyze the relationship between sports and mental health.
* Evaluate the social benefits of sports participation.
* Visualize survey responses using charts and graphs.
* Draw meaningful conclusions from collected data.

## Features

* Survey data analysis
* Data cleaning and preprocessing
* Statistical analysis
* Pie chart visualization
* Bar chart visualization
* Health and lifestyle insights

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Jupyter Notebook

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/sports-impact-analysis.git
cd sports-impact-analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
